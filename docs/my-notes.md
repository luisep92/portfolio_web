# my-notes

Scratchpad for hypotheses, half-formed ideas, and questions that don't yet have a clear home.

**How this file works:**

- Drop notes here freely. Treat it as the cheap space between thinking and committing.
- Tag speculative claims as such (`hypothesis:`, `not yet validated:`) so a future read doesn't mistake them for facts.
- When a note matures into something stable (a decision, an architectural fact, a reusable recipe), **migrate it** to its authoritative home (`DECISIONS.md`, `ARCHITECTURE.md`, the relevant skill) and delete it from here. Don't leave duplicates.
- Per [`docs-governance`](../.claude/skills/docs-governance/SKILL.md), nothing in this file is treated as authoritative.
